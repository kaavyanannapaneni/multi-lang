# i18n example

## Tools
- [i18n](https://www.i18next.com/)
- [React-19.2.3](https://react.dev/)
- [React Bootstrap-2.10.10](https://react-bootstrap.netlify.app/)
- [Vite-7.2.7](https://vite.dev/)

## Installation
```bash
git clone <repo>
cd <project-folder>
npm install
npm run dev 
```

## Walkthrough 
### Setup 
- package.json for libraries 
- src/i18n.js for i18n setup 

## i18n
- Language switching
- Static and dynamic text translation (Inerpolation)
- Trans Component 
- Fallback language support 
- RTL, LTR 
- Lazy loading translation files
